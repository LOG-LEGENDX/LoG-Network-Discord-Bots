    # Minecraft Server Status // Commands FAQ's

    All Commands Used To Get Your Minecraft Server Status 
    Here Are All Commands Which You Can Used To Get Your Minecraft Server Status :
    - Status : Use This Command To Get Status Of Your Minecraft Server
    API Used By Minecraft Server Status Discord Bot 
    Minecraft Server Status Discord Bot Use This API To Fetch Status Your Minecraft Server :
    - Mcsrvstat : Used To Fetch For Both Java And Bedrock Servers , Fetching URL Is Different And Provided By Mcsrvstat
    Players Listing Of Minecraft Server Status Discord Bot 
    If You Use Status-Java Or Status-Bedrock Command , You Can Find These Things In Players Section Of Command Response :
    - Currently Players Are Hidden On This Server : This Message Displayed If Your Minecraft Server Query Is Not True
    - No One Is Currently Playing On This Server : This Message Displayed If Your Minecraft Server Doesn't Have Any PLayer At That Time
    - More Than 10 Players Are Playing , Cant List Them : This Message Displayed If Your Minecraft Server Have More Than 10 PLayers
    - player1 , player2 : This Will Show The Names Of Players In Your Minecraft Server At That Time
