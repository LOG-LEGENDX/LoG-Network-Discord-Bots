    # Minecraft Server Status // Info FAQ's

    All Information Of Minecraft Server Status Discord Bot 
    Minecraft Server Status Bot is a Discord Bot Which Gives You Status Of Your Minecraft Server .
    - Created By : LegendX
    - Supported By : LoG-Network
    All Packages Used By Minecraft Server Status Discord Bot 
    Minecraft Server Status Discord Bot Use These Packages , All Packages Will Be Updated TO Their Newest Version Timely :
    - Discord.JS : v13.6.0
    - FS : v0.0.1-security
    - Glob : v7.2.0
    - Node-Fetch : v3.2.0
    - @DiscordJS/REST : v0.4.1
    - Quick.DB : v7.1.3
    All User Permissions Needed To Operate Minecraft Server Status Discord Bot 
    If You Want To Operate Minecraft Server Status Discord Bot Yourself , You Should Have These Permissions :
    - VIEW_CHANNEL (Its Default)
    - SEND_MESSAGES
    - READ_MESSAGES
    - MANAGE_GUILD
    - READ_MESSAGE_HISTORY
    All Bot Permissions Needed To Operate Minecraft Server Status Discord Bot 
    Minecraft Server Status Discord Bot Needs These Permissions To Run :
    - VIEW_CHANNEL (Its Default)
    - SEND_MESSAGES
    - READ_MESSAGES
    - CREATE_INSTANT_INVITE
    - EMBED_LINKS
    - ATTACH_FILES
    - USE_SLASH_COMMANDS
    - READ_MESSAGE_HISTORY
