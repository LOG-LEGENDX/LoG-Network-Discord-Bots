    # Minecraft Server Status // Config

    # Bot-Setup

    DANGER

    Before Go Into Bot-Config or Bot-Startup Make Sure You Have Downloaded And Installed VISUAL STUDIO CODE and LATEST VERSION OF NODEJS

    NEXT

    After Completing Bot-Setup Move To Bot-Config

    # Bot-Config

    You Have Found File Named config.json like this :

    Click Me To View The Code 
    {
  "websitelink":"your-bot-website-link",
  "docswebsitelink":"your-bot-documentation-website-link",
  "faqswebsitelink":"your-bot-faqs-website-link",
  "botinvitelink": "your-bot-invite-link",
  "bottoken": "your-bot-token",
  "discordinvitelink": "your-discord-server-invite-link",
  "botago": "86400000",
  "botoldcolor": "RED",
  "botnewcolor": "GREEN",
  "botreportchannel": "channel-id-where-report-command-send-reports"
}
    • In First Line Place Your Bot Website Link .
    • In Second Line Place Your Bot Docs Website Link .
    • In Third Line Place Your Bot FAQ's Website Link .
    • In Fourth Line Place Your Bot Invite Link .
    • In Fifth Line Place Your Bot Token .
    • In Sixth Line Place Your Discord Server Invite Link .
    • Don't Make Changes In Seventh , Eighth And Nineth Line .
    • In Last Line Place Discord Channel Id Where Bot Sends A Report Whenever Anyone Used Report Command .

    NEXT

    After Completing Bot-Config Move To Bot-Startup

    # Bot-Startup

    When You Completed The Above Setup Open Terminal And Use These Commands :

    Click Me To View The Code 
    npm install discord.js fs glob node-fetch @discordjs/rest quick.db
npm run start

    Congo Your Bot Is Working Now , Enjoy .
